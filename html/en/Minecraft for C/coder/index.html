Maybe you need to know the meaning of the following signs before you look through the manual:

This is a notice; you need to be aware of it.

This is a warning; you need to read it carefully.

This is a definition, which introduces some technical terms .

This is a parameter list of the function.

This is the other parameter list of the function.

This is a return type of the function.

This is the other return type of the function.

If there're several parameter lists for the function, but there's only one return type, the return type will be shown only once. For example:

This is a parameter list of the function.

This is the other parameter list of the function.

This is the only return type of the function.

Also, if there're several parameter lists as well as return values, they will match one by one. For example:

Parameter list 1.

Parameter list 2.

Parameter list 3.

Return value 1.

Return value 1.

Return value 2.
