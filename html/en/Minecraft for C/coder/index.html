Minecraft for C

Maybe you need to know the meaning of the following signs before you look through the manual: NOTE This is a notice; you need to be aware of it. WARN This is a warning; you need to read it carefully. DEF This is a definition, which introduces some [NOUN technical terms]. LIST input - This is a parameter list of the function. - This is the other parameter list of the function. LIST output - This is a return type of the function. - This is the other return type of the function. If there're several parameter lists for the function, but there's only one return type, the return type will be shown only once. For example: LIST input - This is a parameter list of the function. - This is the other parameter list of the function. LIST output - This is the only return type of the function. Also, if there're several parameter lists as well as return values, they will match one by one. For example: LIST input - Parameter list 1. - Parameter list 2. - Parameter list 3. LIST output - Return value 1. - Return value 1. - Return value 2. If the function doesn't need a parameter or doesn't return anything, it will be displayed like this: LIST input LIST output
