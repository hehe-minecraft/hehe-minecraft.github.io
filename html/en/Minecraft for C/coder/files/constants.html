errors

Here are all the errors which could be thrown in the functions. The inheritence tree of them is right below:

  1. BasicError
    1. ChoiceError
    2. MathError
      1. MatrixDimensionError
      2. VectorDimensionError

BasicError

This is the base of all the user-defined errors.

ChoiceError

It would be thrown if unexpected choices have been chosen.

MathError

It would be thrown if there's a calculation error.

MatrixDimensionError

It would be thrown if the dimensions of the matrixes being calculated don't match.

VectorDimensionError

It would be thrown if the dimensions of the vectors being calculated don't match.
