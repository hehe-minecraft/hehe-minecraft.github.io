Minecraft for C (coder)

Generally speaking, in a matrix multiplication, we put the first matrix on the right side rather on the left side, because it multiplies the vector first. While optimizing the code, it is better to combine all the matrixes before multiplying.

rotate

Build a rotation matrix. Specially, if the inputs are rotating angles in three directions, then the order will be x → y → z .

The parameter axis needs to be among choices::axis_x, choices::axis_y, choices::axis_z , or it will throw a ChoiceError .

choices::axis axis is the rotating axis.
double theta is the rotating angle (clockwise, in degrees).

double theta_x is the rotating angle along x-axis.
double theta_y is the rotating angle along y-axis.
double theta_z is the rotating angle along z-axis.

Matrix is the rotation matrix based on this angle.

scale

Build a scalar matrix.

double ratio is the ratio of scaling.

Matrix is the scalar matrix based on this ratio.

translate

Build a translation matrix.

double x is the translating distance towards x-axis.
double y is the translating distance towards y-axis.
double z is the translating distance towards z-axis.

Matrix is the translation matrix.
