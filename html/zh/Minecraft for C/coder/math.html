Vector

这是一个向量对象。

如果你想参见矩阵和向量相乘，请移步至 Matrix.operator* 。

请在你对于这个向量的最后一个用法之后删除这个向量，因为该向量存在保存在 堆 中的属性。若不按时删除该变量，可能导致 内存泄漏 的严重问题！删除该对象时应使用 delete[] 语句。

const Vector& item 为你要复制的向量。

unsigned short dimensions 定义向量的维度。
double[] values 定义向量的值。

unsigned short dimensions 即向量的维度。
double* values 即向量的值。

operation+

在相加的时候，输入的两个向量必须维度相同，否则会抛出 VectorDimensionError 错误。

double other 表示你要相加的标量对象。

Vector 即相加的结果。
