choices

这里定义了项目中对于函数模式的选择（其实是枚举对象）。

axis

这是各种坐标轴。

值 含义
axis_x x 轴。
axis_y y 轴。
axis_z z 轴。

state

函数或线程的运行状态。

值 含义
state_running 正在运行。
state_free 空闲，等待输入。
state_paused 已暂停。
state_stopped 已终止。

json

Json 对象的存储类型。

值 含义
json_null null 值。
json_bool 布尔值。
json_int 整数值。
json_float 双精度浮点数值。
json_string 字符串。
json_list 列表值。
json_map 字典值。

constants

这里记录了项目中所需要的数学及科学常数。

pi

圆周率，圆周长与半径的比值。

errors

这里定义了项目中所会抛出的所有错误，其层级关系如下：

  1. BasicError
    1. ChoiceError
    2. FileError
      1. FileNotFoundError
    3. IndexError
    4. JsonError
      1. JsonEOFError
      2. JsonSyntaxError
        1. JsonBackSlashError
      3. JsonTypeError
    5. MathError
      1. MatrixDimensionError
      2. VectorDimensionError
    6. NullPointerError
    7. ThreadExistError
    8. UnexpectedControlError

BasicError

这是自定义错误中最基本的错误，所有项目错误都基于它。

ChoiceError

选择的选项在固定选项之外会抛出这个错误。

FileError

在出现文件开关或访问错误的时候会抛出这个错误。

FileNotFoundError

无法找到文件的时候会抛出这个错误。

IndexError

在所取的索引位置不在正确范围内时会抛出这个错误。

JsonBackSlashError

在出现 JSON 字符串中包含意外转义字符时会抛出这个错误。

JsonEOFError

在出现字符串意外终止而 JSON 为解析完成的时候会抛出这个错误。

JsonError

在出现 JSON 问题的时候会抛出这个错误。

JsonTypeError

在出现 JSON 类型无法正确转换的时候会抛出这个错误。

JsonSyntaxError

在出现 JSON 无法正确从字符串中解析的时候会抛出这个错误。

MathError

数学运算出错会抛出这个错误。

MatrixDimensionError

矩阵大小不匹配会抛出这个错误。

NullPointerError

在已知是一个空指针却还要解引用的时候会抛出这个错误。

ThreadExistError

在线程已存在却还要创造线程时会抛出这个错误。

UnexpectedControlError

代码在进入未定义状态的时候会抛出这个错误。

VectorDimensionError

向量维度不匹配会抛出这个错误。
