Json

JSON 存储容器。

如果在构造函数中什么都不输入，会生成存储 null 的对象。

const Json& item 为你要拷贝的 Json 对象。

Json&& item 为你要转移的 Json 对象。

const bool value 为应当存储的布尔值。

const int value 为应当存储的整数值。

const double value 为应当存储的浮点数值。

const std::wstring value 为应当存储的字符串。

const JsonList value 为应当存储的列表。

const JsonMap value 为应当存储的字典。

protected:
choices::json type 即存储的类型。
void* content 即存储的内容指针。

operator[]

获取字典或列表的内容。

如果存储的是字典需传入字符串，列表需传入自然数，如果输入错误，会抛出 JsonTypeError 错误。

你可以输入自然数（从左往右索引）或负数（从右往左索引）。

const int index 为列表的索引。

const std::wstring& index 为字典的索引。

Json 为字典或列表索引处的值。

operator=

Json item 为要改成的 Json 对象。

operator bool

获取存储的布尔值。

隐式转换仅支持存储的布尔值。
显示转换包含隐式转换的内容，并将非 0 的小数和整数、非空的字符串、非空列表和非空字典转化为 true ，将为 0 的小数和整数、空字符串、空列表、空字典和 null 转化为 false 。

bool 为存储的布尔值。

operator int

获取存储的整数。

隐式转换仅支持存储的整数。
显示转换包含隐式转换的内容，并将布尔值的 true 和 false 转化为 1 和 0 ， null 转化为 0 ，小数和字符串转化为整数。

如果无法转换，会抛出 JsonTypeError 错误。

int 为存储的整数。

operator double

获取存储的小数。

隐式转换仅支持存储的小数。
显示转换包含隐式转换的内容，并将布尔值的 true 和 false 转化为 1.0 和 0.0 ， null 转化为 0.0 ，整数和字符串转化为小数。

如果无法转换，会抛出 JsonTypeError 错误。

double 为存储的小数。

operator std::wstring

获取存储的字符串。

隐式转换仅支持存储的字符串。
显示转换包含隐式转换的内容，并将整数、小数、布尔值、 null 都转化为字符串。

如果无法转换，会抛出 JsonTypeError 错误。

std::wstring 为存储的字符串。

operator JsonList

获取存储的列表。

仅支持直接读取。

如果无法转换，会抛出 JsonTypeError 错误。

JsonList 为存储的列表。

operator JsonMap

获取存储的字典。

仅支持直接读取。

如果无法转换，会抛出 JsonTypeError 错误。

JsonMap 为存储的字典。
