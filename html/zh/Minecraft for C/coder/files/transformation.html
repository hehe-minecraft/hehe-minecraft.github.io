一般地，在矩阵乘法中，我们会将先进行变换的矩阵放在右侧而不是左侧，因为它会先乘上向量。在对代码优化的过程中，将矩阵先组合起来在乘上每一个向量是更明智的选择。

rotate

构造一个旋转矩阵。特别地，如果输入的是三个方向的旋转角度，则旋转次序为 x → y → z 。

输入的 axis 参数必须是介于 choices::axis_x, choices::axis_y, choices::axis_z 之间的，否则会抛出 ChoiceError 错误。

choices::axis axis 为旋转轴。
double theta 为旋转角度（顺时针，角度制）。

double theta_x 为绕 x 轴旋转的角度。
double theta_y 为绕 y 轴旋转的角度。
double theta_z 为绕 z 轴旋转的角度。

Matrix 即针对该旋转角度的旋转矩阵。

scale

构造一个放缩矩阵。

double ratio 为放大或缩小的比率。

Matrix 即针对该放缩比率的放缩矩阵。

translate

构造一个位移矩阵。

double x 为向 x 轴正方向平移的距离。
double y 为向 y 轴正方向平移的距离。
double z 为向 z 轴正方向平移的距离。

Matrix 即生成的位移矩阵。
