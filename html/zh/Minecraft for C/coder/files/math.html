Matrix

这是一个矩阵对象。

const Matrix& item 为你要复制的矩阵。

Matrix&& item 为你要转移的矩阵。

const unsigned short[2] dimensions 定义矩阵的维度（先行后列）。

const unsigned short[2] dimensions 定义矩阵的维度（先行后列）。
const double* values 定义指向矩阵的左上角数据的指针。

unsigned short[2] dimensions 即矩阵的维度（先行后列）。
double* values 指向即矩阵左上角数据的指针。

operator+

在相加的时候，输入的两个矩阵必须维度完全相同，否则会抛出 MatrixDimensionError 错误。

const Matrix& other 表示你要相加的矩阵对象。

const double other 表示你要相加的标量对象。

Matrix 即相加的结果。

该运算可以给矩阵取反。

在相减的时候，输入的两个矩阵必须维度完全相同，否则会抛出 MatrixDimensionError 错误。

const Matrix& other 表示你要相减的矩阵对象。

const double other 表示你要相减的标量对象。

Matrix 即相减的结果。

operator*

在相乘的时候，输入的第一个矩阵的列数必须与第二个矩阵的行数相同，否则会抛出 MatrixDimensionError 错误。
如果输入了一个向量，那么第一个矩阵的列数必须与向量的维度相同，否则也会抛出 MatrixDimensionError 错误。

const Matrix& other 表示你要相乘的矩阵对象。

const double other 表示你要相乘的标量对象。

const Vector& other 表示你要相乘的向量对象。

Matrix 即相乘的结果。

Matrix 即相乘的结果。

Vector 即相乘的结果。

operator/

const double other 表示你要相除的标量对象。

Matrix 即相除的结果。

Vector

如果你想参见矩阵和向量相乘，请移步至 Matrix.operator* 。

Vector&& item 为你要转移的x向量。

const unsigned short dimensions 定义向量的维度。
const double[] values 定义向量的值。

unsigned short dimensions 即向量的维度。
double* values 即向量的值。
