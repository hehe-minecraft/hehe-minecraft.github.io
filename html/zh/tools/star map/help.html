    #总体介绍 DEF [NOUN 星图]是一个用来将各条信息整合的工具，就像思维导图一样。这里用于特指本工具。 WARN 这里存储的星图只能在本网站上打开，你可以下载网站的源代码以将这个工具搬运到你的电脑上，但是原先在网站上的内容并不会同步到电脑上。代码在 [GitHub TO BLANK https://github.com/hehe-minecraft/hehe-minecraft.github.io] 开源。 NOTE 本产品灵感来源于 [MCMod TO BLANK https://www.mcmod.cn] 的模组星图功能。 以下是与本产品类似的思维导图软件，也可以去看看： - [Obsidian TO https://obsidian.md] #术语定义 ##节点 DEF [NOUN 节点]是星图中的一个基本单元，存储一片信息。 FIGURE node 节点在显示的时候是一个点（如[FIGURE node]，更准确地说是实心的圆），你可以改变它的颜色、大小、内容。 节点并不一定需要链接起来，也可以不添加任何信息，但是这就没有了星图的意义了。 ##链接 DEF [NOUN 链接]是连接两个节点的线段，可以是单向或者双向的，存储两个节点之间的关系。 FIGURE link 链接在显示的时候是一条线段（如[FIGURE link]），多个链接会共享一种样式，其中包含粗细、方向、颜色。 链接是两种颜色相互交替的，双向的链接表现成间隔打点（[FIGURE link]上半部分），而单向的链接表现成交替箭头（[FIGURE link]下半部分）。 ##元素 DEF 节点和链接统称为[NOUN 元素]。 #使用方法 ##基本操作 点击鼠标左键来选中节点或链接。 长按鼠标左键并拖动来移动星图。 滚动鼠标滚轮来放大或缩小星图。 点击鼠标右键来打开编辑菜单。 ##去鼠标操作 使用键盘同样可以控制星图，使用[KEY Space]来代替对屏幕中心点击鼠标左键，使用[KEY Enter]键来代替对屏幕中心点击鼠标右键，使用[KEY ↑]、[KEY ↓]、[KEY ←]、[KEY →]四个按键来移动星图，使用[KEY =]和[KEY -]对应的位置（不用按住[KEY Shift]）来放大和缩小星图。 WARN 不要同时使用鼠标和键盘进行基本操作，可能有意料之外的错误产生。 ##创建节点 FIGURE node_create_logo 对着背景打开编辑菜单，选择如[FIGURE node_create_logo]的圆形符号来打开节点编辑界面。 在节点编辑界面中，最上端的方块是样式选择区域，在其下方可以填入节点的标题和大小。样式不包含节点大小。 点击样式选择区域最后的“新建”按钮来创建新的样式，它会弹出一个新的界面，其中可以填入样式的名字和颜色。 在编辑完成之后，点击对话框右下角的“确定”以确认编辑。 存在样式时，点击样式选择区域特定的样式来应用它。每个样式会将自己的颜色显示为背景色。 创建节点时标题、大小和样式是必填的。如果没有选择样式，点击“确定”时会在“标题”一框中显示提醒：“你还没有选择样式”。 ##删除节点样式 NOTE 删除节点样式会使得所有使用该样式的节点切换为指定的另一种样式。 在节点编辑界面的样式选择区域中，选中一个样式并按下键盘的[KEY Delete]键来删除样式。 删除样式会出现一个对话框，在这里面选中的样式将作为删除后所有使用该样式的节点的样式。 ##移动节点 FIGURE node_move_logo 对着节点打开编辑菜单，选择如[FIGURE node_move_logo]的箭头符号来移动节点。 在点击这个按钮之后，你需要再左键点击背景的任意位置。你不能将节点移动到另一个节点或链接上。 ##编辑节点 NOTE 编辑节点时，你只能编辑标题、大小和样式，而不是位置和内容。 FIGURE element_edit_logo 对着节点打开编辑菜单，选择如[FIGURE element_edit_logo]的铅笔符号来编辑节点。 编辑节点的界面与创建节点的界面一致。 ##删除节点 NOTE 删除节点时，会同时删除所有与该节点相关的链接。 FIGURE element_delete_logo 对着节点打开编辑菜单，选择如[FIGURE element_delete_logo]的叉号来删除节点。 删除节点没有重新确认界面。 ##创建链接 FIGURE link_create_logo 对着节点打开编辑菜单，选择如[FIGURE link_create_logo]的线段符号来打开链接编辑界面。 在链接编辑界面中，最上端的方块也是样式选择区域，在其下方可以填入链接的标题。样式包含链接粗细。 点击样式选择区域最后的“新建”按钮来创建新的样式，它会弹出一个新的界面，其中可以填入样式的名字、主颜色、次颜色和粗细。“单向”若勾选则显示样式会类似于[FIGURE link]上半部分（单向链接）。 FIGURE link 存在样式时，点击样式选择区域特定的样式来应用它。每个样式会将自己的主颜色显示为字体颜色，将次颜色显示为背景色。 创建链接时标题和样式是必填的。如果没有选择样式，点击“确定”时会在“标题”一框中显示提醒：“你还没有选择样式”。 双向链接无法在两个连接点内反复创建，将它们交换顺序也不行。而对于单向链接，可以将初始点和结束点交换位置再次创建，这样其显示方式会类似于[FIGURE link]下半部分（双向链接）。 ##删除链接样式 删除链接样式的过程与[删除节点样式 TO #使用方法-删除节点样式]的过程类似。 ##编辑链接 NOTE 编辑链接时，你只能编辑标题和样式，而不是内容。 FIGURE element_edit_logo 对着链接打开编辑菜单，选择如[FIGURE element_edit_logo]的铅笔符号来编辑链接。 编辑链接的界面与创建链接的界面一致。 ##删除链接 FIGURE element_delete_logo 对着链接打开编辑菜单，选择如[FIGURE element_delete_logo]的叉号来删除链接。 删除链接没有重新确认界面。 ##编辑内容 FIGURE content_edit_logo 对着节点或链接打开编辑菜单，选择如[FIGURE content_edit_logo]的文本符号来编辑内容。 编辑内容的界面左侧是源代码区域，右侧是预览区域，源代码会在右侧实时预览。 源代码区域中需要输入本网站特有的 Source 代码，但编辑内容部分并不支持所有 Source 代码，支持的代码包括： - 块级元素 - [正文 TO ../../application/source.html#块级元素-正文] - [标题 TO ../../application/source.html#块级元素-标题] - [分隔线 TO ../../application/source.html#块级元素-分隔线] - [列表 TO ../../application/source.html#块级元素-列表] - [注释 TO ../../application/source.html#块级元素-注释] - [代码段 TO ../../application/source.html#块级元素-代码段] - [公式块 TO ../../application/source.html#块级元素-公式块] - [表格 TO ../../application/source.html#块级元素-表格] - 行内元素 - [超链接 TO ../../application/source.html#行内元素-超链接] - [术语 TO ../../application/source.html#行内元素-术语] - [代码片段 TO ../../application/source.html#行内元素-代码片段] - [键盘输入 TO ../../application/source.html#行内元素-键盘输入] - [公式片段 TO ../../application/source.html#块级元素-公式片段] ##自动排版 WARN 自动排版对性能要求较大。 FIGURE auto_typesetting_logo 点击左下角如[FIGURE auto_typesetting_logo]的按钮来打开或关闭自动排版。 自动排版时，节点会缓慢移动，直到它们之间有一个合理的距离，但是常常陷入[NOUN 局部最优 AS 可能不是最好，但是下一步向任意一个方向都会变差]的现象中，此时你需要手动调整节点的位置。 ##搜索 FIGURE search_logo 点击左下角如[FIGURE search_logo]的按钮来打开搜索界面。 你可以看到有两个搜索框，分别是搜索标题和正文。如果不想使用某一个搜索功能，只需要将其留空。两个都留空会为你搜索到任意内容。 下方的复选框可以启用正则表达式，在这里不会讲解正则表达式的语法。 NOTE 搜索到的元素只是“包含”搜索内容，而非全部满足。 你可以选择搜索所有节点还是所有链接。 搜索结果将被选中，星图也会被移动到搜索结果的位置。 对于链接，星图会移动到它两端点的中间。 如果出现多个搜索结果，搜索工具会选中第一个搜索到的。如果你想跳转到下一个，就继续选中这个结果，并再次点击搜索框，确定后就能看见下一个结果。如果这是最后一个搜索结果，确定之后将什么都不发生。 在最开始的搜索前，你需要先选中背景，否则搜索工具会误以为你要从这里继续搜索。 ##保存 WARN 在编辑星图的状态下切换星图或者退出星图不会自动保存！ FIGURE save_logo 点击左下角如[FIGURE save_logo]的按钮就可以保存。保存的文件名称与打开星图时的文件名称相同。 在打开某一星图的状态下离开网站会提示注意保存。 ##撤销与重做 撤销与重做没有专门的按钮，按下键盘快捷键[KEY Ctrl][KEY Z]进行撤销，按下键盘快捷键[KEY Ctrl][KEY Y]进行重做。 DEF 在大多数改动时都会生成一个[NOUN 快照]，其包含了此时的星图内容。 以下改动不会生成快照： - [移动星图 TO #使用方法-基本操作] - [自动排版造成的节点移动 TO #使用方法-自动排版] 在历史快照进行任何操作会抹去该快照之后所有的快照。 快照不仅会随着星图保存，仅限改动时有效。 #常见问题 ##卡在星图里面出不来 假如你通过非星图区域放大了屏幕，又用鼠标滚轮滑进了星图区域，再进行鼠标滚轮就无法逃脱，因为此时系统判定你正在放大或缩小星图，此时就会导致整个屏幕都是星图区域而无法正常执行操作。 要解决这个问题，你可以在星图区域上使用键盘的[KEY ↑]、[KEY ↓]、[KEY ←]、[KEY →]键或使用[KEY PageDown]和[KEY PageUp]键。 ##星图无法打开，需要更新 在旧版本创建的星图可能不可以在新版本使用，反之亦然。你需要前往[更新界面 TO update.html]去升级或降级星图版本。
