简介

在这里可以升级或降级你的星图版本。

升级或降级星图版本操作不可逆，降级后的星图可能出现部分特性消失的问题。

这里使用基于 Typescript Interface 的扩展语法进行标注。

下表列出了版本星图结构的变量含义：

变量 含义
File 整个星图文件
Node 节点
Link 链接
NodeStyle 节点样式
LinkStyle 链接样式
